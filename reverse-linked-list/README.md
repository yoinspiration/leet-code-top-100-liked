# Reverse Linked List

反转链表

## 题目要求

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

**示例：**

```
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]

输入：head = [1,2]
输出：[2,1]

输入：head = []
输出：[]
```

## 解法一：迭代反转 [Iterative.cpp](Iterative.cpp)

### 算法思路

**核心思想：三指针迭代反转**

维护三个指针：
- `prev`：已反转部分的头结点（初始为 `nullptr`）
- `cur`：当前正在处理的节点
- `next`：暂存 `cur->next`，防止链表断开

对每个节点 `cur`：
1. 暂存下一节点：`next = cur->next`
2. 反转指针：`cur->next = prev`
3. 推进指针：`prev = cur`, `cur = next`

循环结束后：
- `cur == nullptr`，遍历完原链表
- `prev` 指向新链表的头结点

### 复杂度分析

- **时间复杂度：O(n)**  
  遍历链表一次。

- **空间复杂度：O(1)**  
  只使用了常数级额外指针。

### 解法评价

迭代法是反转链表最常用、最直观的写法，配合三指针可以清晰地控制指针方向和遍历过程。也可以用递归实现，但递归会使用 O(n) 的栈空间，通常迭代法更优。
