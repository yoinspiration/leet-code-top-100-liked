# Set Matrix Zeroes

矩阵置零

## 题目要求

给定一个 `m x n` 的矩阵，如果一个元素为 `0`，则将其所在行和列的所有元素都设为 `0`。

**要求：** O(1) 空间复杂度（原地操作）。

**示例：**

```
输入：matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出：[[1,0,1],[0,0,0],[1,0,1]]

输入：matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
输出：[[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

## 解法：原地标记 [InPlace.cpp](InPlace.cpp)

### 算法思路

**核心思想：** 使用第一行和第一列作为标记，记录哪些行和列需要置零。

**关键问题：** 第一行和第一列本身也可能需要置零，所以需要额外记录。

**算法步骤：**

1. **记录第一行和第一列的状态**
   - `firstRowZero`：第一行是否有 0
   - `firstColZero`：第一列是否有 0

2. **使用第一行和第一列作为标记**
   - 遍历 `matrix[1..m-1][1..n-1]`
   - 如果 `matrix[i][j] == 0`，则：
     - `matrix[i][0] = 0`（标记第 i 行需要置零）
     - `matrix[0][j] = 0`（标记第 j 列需要置零）

3. **根据标记置零**
   - 遍历 `matrix[1..m-1][1..n-1]`
   - 如果 `matrix[i][0] == 0` 或 `matrix[0][j] == 0`，则 `matrix[i][j] = 0`

4. **处理第一行和第一列**
   - 如果 `firstRowZero == true`，将第一行全部置零
   - 如果 `firstColZero == true`，将第一列全部置零

**示例：**

```
输入：
[[1, 1, 1],
 [1, 0, 1],
 [1, 1, 1]]

步骤1：检查第一行和第一列
- firstRowZero = false
- firstColZero = false

步骤2：标记（发现 matrix[1][1] == 0）
- matrix[1][0] = 0（标记第1行）
- matrix[0][1] = 0（标记第1列）
矩阵变为：
[[1, 0, 1],
 [0, 0, 1],
 [1, 1, 1]]

步骤3：根据标记置零
- matrix[0][1] == 0 → 第1列置零
- matrix[1][0] == 0 → 第1行置零
矩阵变为：
[[1, 0, 1],
 [0, 0, 0],
 [1, 0, 1]]

步骤4：第一行和第一列不需要置零（因为 firstRowZero 和 firstColZero 都是 false）
最终结果：
[[1, 0, 1],
 [0, 0, 0],
 [1, 0, 1]]
```

### 复杂度分析

- **时间复杂度：O(m × n)**
  - 需要遍历矩阵两次
- **空间复杂度：O(1)**
  - 只使用了两个布尔变量（`firstRowZero`、`firstColZero`）

### 解法评价

最优解法，时间复杂度 O(m × n)，空间复杂度 O(1)。通过巧妙地使用第一行和第一列作为标记，避免了使用额外的 O(m + n) 空间。
