# Maximum Subarray

最大子数组和

## 题目要求

给你一个整数数组 `nums`，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

**子数组** 是数组中的一个连续部分。

**示例：**

```
输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6。

输入：nums = [1]
输出：1

输入：nums = [5,4,-1,7,8]
输出：23
```

## 解法一：动态规划（Kadane算法）[DynamicProgramming.cpp](DynamicProgramming.cpp)

### 算法思路

使用动态规划的思想，定义状态：
- `dp[i]`：以位置 `i` 结尾的最大子数组和

**状态转移方程：**
```
dp[i] = max(nums[i], dp[i-1] + nums[i])
```

**解释：**
- 如果 `dp[i-1] > 0`，继续累加可能得到更大的和
- 如果 `dp[i-1] <= 0`，重新开始（只取 `nums[i]`）会更好

**空间优化：**
- 由于 `dp[i]` 只依赖 `dp[i-1]`，可以用一个变量 `currentSum` 代替数组
- 同时用一个变量 `maxSum` 记录全局最大值

### 复杂度分析

- **时间复杂度：O(n)**
  - 只需一次遍历数组
- **空间复杂度：O(1)**
  - 只使用了常数额外空间

### 解法评价

最优解法，时间复杂度 O(n)，空间复杂度 O(1)。这是经典的 **Kadane 算法**，是解决最大子数组和问题的标准解法。

## 解法二：暴力枚举 [BruteForce.cpp](BruteForce.cpp)

### 算法思路

双重循环枚举所有可能的连续子数组，计算每个子数组的和，找出最大值。

### 复杂度分析

- **时间复杂度：O(n²)**
  - 外层循环：O(n)
  - 内层循环：O(n)
  - 每个子数组的和计算：O(1)（可以累加计算）
- **空间复杂度：O(1)**
  - 只使用了常数额外空间

### 解法评价

**注意：暴力枚举方法可能无法通过本题（会超时）。** 由于时间复杂度为 O(n²)，LeetCode 的测试用例中包含大规模数据，可能会导致超时。思路简单但效率低，仅作学习参考。实际提交时**必须使用动态规划优化版本**。
