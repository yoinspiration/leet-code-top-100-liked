# Number of Islands

岛屿数量

## 题目要求

给定一个由 `'1'`（陆地）和 `'0'`（水）组成的二维网格，计算其中**岛屿的数量**。

岛屿由若干相邻（上下左右，不含对角线）的陆地单元格组成，四周被水包围。

题目链接：[200. 岛屿数量](https://leetcode.cn/problems/number-of-islands/)

## 解法：DFS 染色 [DFS.cpp](DFS.cpp)

遍历整个网格：遇到 `'1'` 时岛屿数 +1，并立即用 DFS 将该岛屿所有连通的 `'1'` 原地改写为 `'0'`（"淹没"处理），防止重复计数。

**关键点：**
- 原地修改代替 `visited` 数组，节省 O(m×n) 额外空间。
- DFS 四个方向递归，越界或遇 `'0'` 直接返回。

- **时间复杂度：O(m×n)**，每个格子最多访问一次。
- **空间复杂度：O(m×n)**，最坏情况递归栈深度（全为陆地时退化为 O(m×n)）。
