# Perfect Squares

完全平方数

## 题目要求

给你一个整数 `n`，返回和为 `n` 的完全平方数的最少数量。

**完全平方数** 是一个整数，其值等于另一个整数的平方（例如，`1`、`4`、`9`、`16`）。

**示例：**

```
输入：n = 12
输出：3
解释：12 = 4 + 4 + 4

输入：n = 13
输出：2
解释：13 = 4 + 9
```

## 解法：动态规划 [DynamicProgramming.cpp](DynamicProgramming.cpp)

### 算法思路

定义 `dp[i]` 为和为 `i` 所需的最少完全平方数个数，初始化 `dp[0] = 0`，其余为正无穷。

对于每个 `i`，枚举所有满足 `j*j <= i` 的完全平方数 `j²`：

```
dp[i] = min(dp[i - j*j] + 1)
```

本质上是一个**完全背包**问题：物品为各完全平方数，可重复选取，求恰好装满背包的最少件数。

### 复杂度分析

- **时间复杂度：O(n√n)**
  - 外层枚举 `i` 共 n 次，内层枚举 `j` 至多 √n 次
- **空间复杂度：O(n)**
  - dp 数组大小为 n+1
