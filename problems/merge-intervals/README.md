# Merge Intervals

合并区间

## 题目要求

以数组 `intervals` 表示若干个区间的集合，其中单个区间为 `intervals[i] = [starti, endi]`。请你合并所有重叠的区间，并返回 **一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间**。

**示例：**

```
输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
输出：[[1,6],[8,10],[15,18]]
解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6]。

输入：intervals = [[1,4],[4,5]]
输出：[[1,5]]
解释：区间 [1,4] 和 [4,5] 可被视为重叠区间。
```

## 解法：排序 [Sorting.cpp](Sorting.cpp)

### 算法思路

1. **排序**：按区间的左端点（`start`）从小到大排序
   - 排序后，重叠的区间会相邻
   - 例如：`[[1,3],[2,6],[8,10]]` 排序后还是 `[[1,3],[2,6],[8,10]]`

2. **合并**：遍历排序后的区间
   - 如果当前区间的左端点 <= 结果中最后一个区间的右端点，说明重叠
   - 合并：更新最后一个区间的右端点为 `max(最后一个区间的右端点, 当前区间的右端点)`
   - 如果不重叠，将当前区间加入结果

**关键点：**
- 排序后，只需要比较当前区间和结果中的最后一个区间
- 如果 `current[0] <= last[1]`，说明重叠
- 合并时取右端点的最大值：`last[1] = max(last[1], current[1])`

### 复杂度分析

- **时间复杂度：O(n log n)**
  - 排序：O(n log n)
  - 遍历合并：O(n)
  - 总体：O(n log n)
- **空间复杂度：O(n)**
  - 结果数组最多存储 n 个区间
  - 如果考虑排序的空间复杂度，则为 O(log n)（递归栈）

### 解法评价

最优解法，时间复杂度 O(n log n)。通过排序将问题转化为简单的线性合并，思路清晰，实现简单。

### 示例演示

```
输入：intervals = [[1,3],[2,6],[8,10],[15,18]]

步骤1：排序（已经按左端点排序）
  [[1,3],[2,6],[8,10],[15,18]]

步骤2：合并
  result = [[1,3]]
  
  i=1: [2,6] 与 [1,3] 重叠（2 <= 3）
       合并为 [1,6]
       result = [[1,6]]
  
  i=2: [8,10] 与 [1,6] 不重叠（8 > 6）
       加入结果
       result = [[1,6],[8,10]]
  
  i=3: [15,18] 与 [8,10] 不重叠（15 > 10）
       加入结果
       result = [[1,6],[8,10],[15,18]]

输出：[[1,6],[8,10],[15,18]]
```
