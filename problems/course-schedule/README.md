# Course Schedule

课程表

## 题目要求

共有 `numCourses` 门课，编号 `0` 到 `numCourses - 1`。给定先修关系数组 `prerequisites`，其中 `[a, b]` 表示学课程 `a` 前必须先学课程 `b`。判断是否能够完成所有课程。

本质：**有向图中是否存在环**。

题目链接：[207. 课程表](https://leetcode.cn/problems/course-schedule/)

## 解法：拓扑排序 BFS（Kahn 算法）[TopoSortBFS.cpp](TopoSortBFS.cpp)

有向无环图（DAG）才存在拓扑序，因此检测能否拓扑排序等价于检测是否无环。

**步骤：**
1. 建邻接表，统计每个结点的**入度**。
2. 将所有入度为 0 的结点入队（无前置依赖，可直接学）。
3. 每次出队一个结点，将其所有后继的入度 -1；若后继入度变为 0，则入队。
4. 统计出队总数：若等于 `numCourses`，说明所有结点都被处理，图无环；否则有环（环内结点入度永远不为 0，永远不会入队）。

- **时间复杂度：O(V + E)**，V 为课程数，E 为先修关系数。
- **空间复杂度：O(V + E)**，邻接表 + 入度数组 + 队列。
