# Group Anagrams

字母异位词分组

## 题目要求

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 是由重新排列源单词的字母得到的一个新单词，所有源单词中的字母通常恰好只用一次。

**示例：**

```
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
输出: [["bat"],["nat","tan"],["ate","eat","tea"]]
```

## 解法：哈希表 [HashTable.cpp](HashTable.cpp)

### 算法思路

使用哈希表将每个字符串排序后的结果作为键，原始字符串作为值的列表。字母异位词在排序后会得到相同的字符串，从而被归入同一组。

1. 遍历输入字符串数组
2. 对每个字符串进行排序，将排序后的字符串作为键
3. 将原始字符串添加到对应的值列表中
4. 遍历哈希表，将所有分组收集到结果中

### 复杂度分析

- **时间复杂度：O(n \* k log k)**
  - n 是字符串数组的长度
  - k 是字符串的平均长度
  - 对每个字符串排序的时间复杂度为 O(k log k)
  - 哈希表操作的时间复杂度为 O(1)
- **空间复杂度：O(n \* k)**
  - 哈希表最多存储 n 个字符串（排序后的键和原始字符串）

### 解法评价

思路清晰，实现简单。排序操作是时间复杂度的主要瓶颈。
