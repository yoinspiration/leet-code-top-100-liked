# Intersection of Two Linked Lists

相交链表

## 题目要求

给你两个单链表的头节点 `headA` 和 `headB`，请你找出并返回两个单链表相交的起始节点。

如果两个链表不存在相交节点，返回 `nullptr`。

**注意：**
- 题目中链表**可能带环或不带环**（本题默认无环版本）
- 相交是指**节点地址相同**，不是值相同

## 解法：双指针对齐长度 [TwoPointers.cpp](TwoPointers.cpp)

### 算法思路

**核心思想：** 使用两根指针分别遍历两条链表，通过「切换头节点」的方式，让两根指针走过的总路程相同，从而在相交点相遇。

设两条链表的长度分别为：
- 不相交部分：`a`、`b`
- 共同相交部分：`c`

两条链表的总长度分别为：`a + c` 和 `b + c`。

如果我们让：
- 指针 `pA` 先走完 `a + c`，然后接着走链表 B 的 `b + c`
- 指针 `pB` 先走完 `b + c`，然后接着走链表 A 的 `a + c`

那么当它们都走了 `a + b + c` 步之后：
- 如果存在相交点，它们会在相交点相遇
- 如果不存在相交点，它们会同时走到 `nullptr`

### 算法步骤

1. 用两个指针 `pA`、`pB`，分别指向 `headA`、`headB`
2. 不断前进，直到 `pA == pB`：
   - 如果 `pA` 走到尾部，则跳到 `headB`
   - 如果 `pB` 走到尾部，则跳到 `headA`
3. 当 `pA == pB` 时：
   - 如果相交，则为相交节点
   - 如果不相交，则为 `nullptr`

### 复杂度分析

- **时间复杂度：O(a + b + c)**，本质 O(m + n)
- **空间复杂度：O(1)**，只使用了常数额外空间
