# Maximum Depth of Binary Tree

二叉树的最大深度

## 题目要求

给定一棵二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点个数。

说明：叶子节点是指没有子节点的节点。

## 解法一：递归（后序思想）[Recursive.cpp](Recursive.cpp)

空树深度为 `0`；否则深度为 `1 + max(左子树深度, 右子树深度)`。

- **时间复杂度：O(n)**，每个结点访问一次。
- **空间复杂度：O(h)**，递归栈深度为树高；最坏链状 `O(n)`。

## 解法二：层序遍历（BFS）[BFS.cpp](BFS.cpp)

用队列按层扩展，每处理完一层深度加一。

- **时间复杂度：O(n)**
- **空间复杂度：O(w)**，`w` 为单层最宽结点数，最坏 `O(n)`。

### 解法评价

递归代码最短；BFS 思路直观（一层一层数）。二者时间都是线性。
