# Binary Tree Level Order Traversal

二叉树的层序遍历

## 题目要求

给你二叉树的根结点 `root`，返回其结点值的层序遍历结果（即逐层地，从左到右访问所有结点）。

## 解法一：层序遍历（BFS）[BFS.cpp](BFS.cpp)

用队列维护当前层结点。每轮先记录队列大小 `levelSize`，再弹出 `levelSize` 个结点并收集值；同时把下一层子结点入队。

- **时间复杂度：O(n)**，每个结点仅入队、出队一次。
- **空间复杂度：O(w)**，`w` 为树的最大层宽，最坏 `O(n)`。

### 解法评价

这是该题最直观、最常见的解法，能够自然地按层组织答案。
