# Add Two Numbers

两数相加

## 题目要求

给你两个非空链表，表示两个非负整数；数字最高位在链表末尾，每位数字按逆序存储，并且每个节点只存储一位数字。请你将两数相加，并以相同形式返回一条表示和的链表。

**示例：**

```
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807

输入：l1 = [0], l2 = [0]
输出：[0]

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]
```

## 解法：哑结点 + 逐位进位 [Iterative.cpp](Iterative.cpp)

### 算法思路

**核心思想：** 与手算加法相同，从低位到高位（链表头到尾）逐位相加。

- 用整数 `carry` 保存进位（0 或 1；本题两数字每位 0–9，进位最多为 1）。
- 每一轮：`sum = carry + (l1 当前位，若有) + (l2 当前位，若有)`，新结点值为 `sum % 10`，下一轮 `carry = sum / 10`。
- 用哑结点 `dummy` 挂住结果链表，`tail` 指向当前末尾，便于尾插新结点。
- 循环条件需包含 `carry != 0`，以处理最高位再进位（例如 `5 + 5 = 10` 要多一位）。

### 复杂度分析

- **时间复杂度：O(max(m, n))**  
  `m`、`n` 为两条链表长度，每位最多处理一次。

- **空间复杂度：O(max(m, n))**  
  结果链表长度最多为 `max(m, n) + 1`（含新分配的结点）。

### 解法评价

迭代模拟是标准写法；若题目要求不新建结点而只改指针，会明显不同，但 LeetCode 本题通常允许新建结果结点。
