# Copy List with Random Pointer

随机链表的复制

## 题目要求

给你一个长度为 `n` 的链表，每个节点包含两个指针：

- `next`：指向下一个节点
- `random`：随机指向链表中的任意节点或 `null`

请构造这个链表的深拷贝，返回新链表头节点。  
新链表中节点值与原链表一致，但所有指针都必须指向新链表中的节点，不能引用原链表节点。

## 解法：哈希表映射 [HashMap.cpp](HashMap.cpp)

### 算法思路

**核心思想：** 用哈希表记录原节点到新节点的一一映射。

- 第一遍遍历：只创建新节点，建立 `oldNode -> newNode` 映射。
- 第二遍遍历：通过映射补全每个新节点的 `next` 和 `random`。
  - `newNode->next = map[oldNode->next]`（若为空则置空）
  - `newNode->random = map[oldNode->random]`（若为空则置空）

这样可保证无论 `random` 指向前面、后面还是自身，都能正确关联到对应副本节点。

### 复杂度分析

- **时间复杂度：O(n)**，两次线性遍历。
- **空间复杂度：O(n)**，哈希表存储 `n` 个节点映射。

### 解法评价

这是最直观且稳定的写法。还存在 O(1) 额外空间的「原地穿插复制」解法，但实现更复杂。
