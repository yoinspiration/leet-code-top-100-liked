# Binary Tree Inorder Traversal

二叉树的中序遍历

## 题目要求

给定一棵二叉树的根节点 `root`，返回其节点值的**中序**遍历结果。

中序遍历顺序：**左子树 → 根 → 右子树**。

## 解法一：递归 [Recursive.cpp](Recursive.cpp)

按定义递归：先左、再访问根、再右。

- **时间复杂度：O(n)**，每个节点访问一次。
- **空间复杂度：O(h)**，递归栈深度为树高 `h`；最坏链状为 `O(n)`。

## 解法二：迭代 + 栈 [Iterative.cpp](Iterative.cpp)

用栈显式模拟递归路径：

1. 当前节点沿左子树一直入栈，直到左子为空。
2. 弹出栈顶，访问其值，再令当前节点为其右孩子。
3. 重复直到栈空且当前为空。

- **时间复杂度：O(n)**
- **空间复杂度：O(h)**，栈中最多存一条左链上的节点。

### 解法评价

递归最直观；迭代适合需要控制栈深度或改写为显式状态机的场景。另有 Morris 遍历可达 `O(1)` 辅助空间，但实现更绕。
