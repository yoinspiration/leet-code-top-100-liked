# Jump Game II

跳跃游戏 II

## 题目要求

给定一个长度为 `n` 的整数数组 `nums`，你最初位于**下标 0**。每个元素 `nums[i]` 表示从下标 `i` 出发可以跳跃的**最大长度**。

返回到达下标 `n - 1` 的**最少跳跃次数**（保证总是可以到达）。

**示例：**

```
输入：nums = [2,3,1,1,4]
输出：2
解释：跳到下标 1（值为 3），再跳到最后一个下标，共 2 次。

输入：nums = [2,3,0,1,4]
输出：2
```

## 解法：贪心 [Greedy.cpp](Greedy.cpp)

### 算法思路

维护两个变量：
- `curEnd`：当前这一跳能覆盖的最远边界
- `maxReach`：从已访问位置出发所能到达的全局最远下标

遍历到 `i == curEnd` 时，说明当前这跳的所有位置都已考察完毕，**必须再跳一次**，将 `curEnd` 更新为 `maxReach`（选最优落点）。

循环只到 `n - 2`，因为到达最后一个下标时不需要再跳。

### 复杂度分析

- **时间复杂度：O(n)** — 单次遍历
- **空间复杂度：O(1)** — 只用常数额外空间

### 解法评价

最优解法。贪心策略：每次跳跃时不关心具体落点，只关心下一跳能覆盖的最远边界，从而保证跳跃次数最少。
