# 78. 子集

https://leetcode.cn/problems/subsets/

## 题目

给你一个整数数组 `nums`，数组中的元素**互不相同**。返回该数组所有可能的子集（幂集）。解集不能包含重复的子集，可以按**任意顺序**返回。

**示例：**
```
输入：nums = [1,2,3]
输出：[[], [1], [1,2], [1,2,3], [1,3], [2], [2,3], [3]]
```

## 思路：回溯

与全排列的区别：
- 全排列：每层从**所有未选**元素中挑一个，路径长度等于 n 时才记录
- 子集：每层从 **start 之后**的元素中挑，**每进入递归就记录**当前路径

```
backtrack(start=0, path=[])   → 记录 []
  选1 → backtrack(start=1, path=[1])  → 记录 [1]
    选2 → backtrack(start=2, path=[1,2])  → 记录 [1,2]
      选3 → backtrack(start=3, path=[1,2,3]) → 记录 [1,2,3]
    选3 → backtrack(start=3, path=[1,3])  → 记录 [1,3]
  选2 → backtrack(start=2, path=[2])  → 记录 [2]
    选3 → backtrack(start=3, path=[2,3])  → 记录 [2,3]
  选3 → backtrack(start=3, path=[3])  → 记录 [3]
```

- **start** 的作用：保证只往后选，避免 `[1,2]` 和 `[2,1]` 重复
- **撤销**：`path.pop_back()` 回到上一状态，换下一个元素继续

## 复杂度

- 时间：O(n × 2ⁿ)，共 2ⁿ 个子集，每个子集平均长度 n/2
- 空间：O(n)，递归栈深度
