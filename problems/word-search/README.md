# 79. 单词搜索

https://leetcode.cn/problems/word-search/

## 题目

给定一个 `m x n` 二维字符网格 `board` 和一个字符串 `word`。如果 `word` 存在于网格中，返回 `true`；否则返回 `false`。

单词必须按照字母顺序，通过相邻单元格（上下左右）构成；同一个单元格在一次路径中不能重复使用。

**示例：**
```
输入：board = [["A","B","C","E"],["S","F","C","S"],["A","D","E","E"]], word = "ABCCED"
输出：true
```

## 思路：回溯 + DFS

把每个格子都当作起点尝试匹配 `word`：

- 若当前格子字符不等于 `word[index]`，当前分支失败
- 若匹配成功，则继续向四个方向搜索下一字符
- 搜索前把当前格子临时标记（如 `'#'`），防止同一路径重复使用
- 回溯返回时恢复字符，保证不影响其他起点/分支

```
从 board[r][c] 开始匹配 word[index]
1) 越界 / 字符不等 -> false
2) index 到末尾 -> true
3) 标记当前格子已访问
4) 递归搜索上下左右的 index + 1
5) 恢复当前格子
```

## 复杂度

- 时间：O(m x n x 3^L)，`L` 为 `word` 长度（首步最多 4 个方向，后续最多 3 个）
- 空间：O(L)，递归栈深度
