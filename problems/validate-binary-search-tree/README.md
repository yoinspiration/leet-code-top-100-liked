# Validate Binary Search Tree

验证二叉搜索树

## 题目要求

给定一棵二叉树的根结点，判断其是否为**有效的二叉搜索树**。

有效 BST 定义：若结点的左子树不为空，则左子树上**所有**结点的值均**小于**它的根结点值；若右子树不为空，则右子树上**所有**结点的值均**大于**它的根结点值；左右子树也分别是二叉搜索树。

题目链接：[98. 验证二叉搜索树](https://leetcode.cn/problems/validate-binary-search-tree/)

## 解法一：递归 + 上下界 [RecursiveBounds.cpp](RecursiveBounds.cpp)

每个结点携带合法区间 \((\text{low}, \text{high})\)（开区间）。当前值须在区间内；向左递归时上界变为当前值，向右递归时下界变为当前值。上下界用 `long long`，避免与 `INT_MIN`/`INT_MAX` 边界比较时整型溢出。

- **时间复杂度：O(n)**，每个结点访问一次。
- **空间复杂度：O(h)**，递归栈为树高。

## 解法二：中序遍历 [Inorder.cpp](Inorder.cpp)

BST 的中序遍历为**严格递增**序列。递归中序访问时维护前一个值 `prev`，若出现 `当前值 <= prev` 则不是 BST。

- **时间复杂度：O(n)**
- **空间复杂度：O(h)**，递归栈；若改为显式栈迭代，仍为 O(h) 辅助空间。

### 解法评价

上下界 DFS 最直接对应 BST 定义；中序法适合已熟悉「BST 与中序有序」的场景。二者线性时间等价，选其一即可。
