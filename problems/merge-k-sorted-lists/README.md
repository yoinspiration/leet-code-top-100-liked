# Merge k Sorted Lists

合并 K 个升序链表

## 题目要求

给你一个链表数组 `lists`，每个链表都已经按升序排列。  
请将所有链表合并到一个升序链表中并返回。

## 解法：最小堆（优先队列） [PriorityQueue.cpp](PriorityQueue.cpp)

### 算法思路

把每条链表当前还未合并部分的头结点看作一个候选最小值，用最小堆维护这些候选：

1. 先将所有非空链表头结点入堆。
2. 每次从堆顶弹出最小节点，接到结果链表尾部。
3. 若该节点有后继，把后继入堆。
4. 堆为空时，合并结束。

因为堆里最多同时有 `k` 个节点（每条链表最多贡献一个“当前头”），每次弹出/插入的代价是 `O(log k)`。

### 复杂度分析

- 设总节点数为 `N`，链表条数为 `k`。
- **时间复杂度：O(N log k)**，每个节点最多入堆、出堆各一次。
- **空间复杂度：O(k)**，堆中最多 `k` 个节点。

### 解法评价

这是本题最常见且效率稳定的写法。另一种常见方案是分治两两合并，时间复杂度同为 `O(N log k)`。
