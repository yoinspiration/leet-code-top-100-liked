# Daily Temperatures

## 解法：单调栈 [Stack.cpp](Stack.cpp)

### 算法思路

用一个**单调递减栈**存储尚未找到更高温度的天的下标。

遍历每一天 `i`：
- 若当前温度高于栈顶下标对应的温度，说明找到了答案：弹出栈顶 `j`，`answer[j] = i - j`
- 重复上述操作直到栈为空或栈顶温度 ≥ 当前温度
- 将 `i` 压栈

以 `[73,74,75,71,69,72,76,73]` 为例：

```
i=0 (73): st=[0]
i=1 (74): 74>73 → answer[0]=1, st=[1]
i=2 (75): 75>74 → answer[1]=1, st=[2]
i=3 (71): st=[2,3]
i=4 (69): st=[2,3,4]
i=5 (72): 72>69 → answer[4]=1, 72>71 → answer[3]=2, st=[2,5]
i=6 (76): 76>72 → answer[5]=1, 76>75 → answer[2]=4, st=[6]
i=7 (73): st=[6,7]
结果: [1,1,4,2,1,1,0,0]
```

### 复杂度分析

- **时间复杂度：O(n)**，每个下标最多入栈、出栈各一次
- **空间复杂度：O(n)**，栈最多存储 n 个下标
