# Binary Tree Right Side View

二叉树的右视图

## 题目要求

给定二叉树根结点 `root`，返回从右侧看到的结点值，按从上到下顺序组成数组。

题目链接：[199. 二叉树的右视图](https://leetcode.cn/problems/binary-tree-right-side-view/description/)

## 解法一：层序遍历（BFS）[BFS.cpp](BFS.cpp)

逐层遍历二叉树。对于每一层，最后处理到的结点就是这一层从右侧能看到的结点，将其加入答案。

- **时间复杂度：O(n)**，每个结点仅入队出队一次。
- **空间复杂度：O(w)**，`w` 为树的最大层宽，最坏 `O(n)`。

## 解法二：深度优先搜索（右优先）[DFS.cpp](DFS.cpp)

按照 `根 -> 右 -> 左` 的顺序 DFS，并记录当前深度。若某一深度是第一次访问，当前结点即该层最右可见结点。

- **时间复杂度：O(n)**。
- **空间复杂度：O(h)**，递归栈深度为树高，最坏 `O(n)`。

### 解法评价

BFS 更直观（按层取末尾）；DFS 更精炼（按深度首次命中）。两者都为线性时间。
