# 51. N 皇后

https://leetcode.cn/problems/n-queens/

## 题目

按照国际象棋的规则，皇后可以攻击同一行、同一列、同一条斜线上的棋子。  
给你一个整数 `n`，返回所有不同的 `n` 皇后问题解法。  

每一种解法包含一个不同的棋盘放置方案，其中：
- `'Q'` 表示皇后
- `'.'` 表示空位

**示例：**
```
输入：n = 4
输出：[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]
```

## 思路：回溯 + 剪枝

逐行放皇后：在第 `row` 行尝试每一列 `col`，只要这个位置不与之前行的皇后冲突，就可以放置并进入下一行。

为了 O(1) 判断冲突，维护三个集合（或布尔数组）：
- `cols[col]`：列 `col` 是否已被占用
- `diag1[row - col + (n - 1)]`：主对角线（左上到右下）是否被占用
- `diag2[row + col]`：副对角线（右上到左下）是否被占用

当 `row == n` 时，说明前 `n` 行都合法放置完成，记录当前棋盘。

```
backtrack(row=0)
  尝试 col=0..n-1
    若列/对角线可用:
      放置 Q，标记占用
      backtrack(row+1)
      撤销 Q，取消标记
```

## 复杂度

- 时间：O(n!)，每一行选择会不断被剪枝，最坏接近全排列规模
- 空间：O(n²)，棋盘存储 O(n²)，递归与标记数组 O(n)
