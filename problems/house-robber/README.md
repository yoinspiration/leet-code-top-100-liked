# House Robber

打家劫舍

## 题目要求

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警**。

给定一个整数数组 `nums`，返回你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

**示例：**

```
输入：nums = [2,7,9,3,1]
输出：12
解释：偷窃第 1、3、5 间房屋，金额 = 2 + 9 + 1 = 12
```

## 解法：动态规划 [DynamicProgramming.cpp](DynamicProgramming.cpp)

### 算法思路

定义 `dp[i]` 为考虑前 `i` 间房屋时能偷到的最高金额。

对于第 `i` 间房屋，只有两种选择：

1. **不偷第 i 间**：`dp[i] = dp[i-1]`
2. **偷第 i 间**：则不能偷第 `i-1` 间，`dp[i] = dp[i-2] + nums[i]`

因此状态转移方程为：

```
dp[i] = max(dp[i-1], dp[i-2] + nums[i])
```

空间优化：每次只依赖前两个状态，用两个变量滚动更新即可。

### 复杂度分析

- **时间复杂度：O(n)**
  - 只需一次遍历
- **空间复杂度：O(1)**
  - 只使用两个变量存储前两个状态
