# Climbing Stairs

爬楼梯

## 题目要求

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例：**

```
输入：n = 3
输出：3
解释：有三种方法可以爬到楼顶。
1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶
```

## 解法：动态规划 [DynamicProgramming.cpp](DynamicProgramming.cpp)

### 算法思路

定义 `dp[i]` 为爬到第 `i` 阶楼梯的不同方案数。

到达第 `i` 阶只有两种情况：

1. 从第 `i-1` 阶爬 1 步上来
2. 从第 `i-2` 阶爬 2 步上来

因此状态转移方程为：

```
dp[i] = dp[i-1] + dp[i-2]
```

这本质上是 Fibonacci 数列，初始值 `dp[1] = 1`，`dp[2] = 2`。

空间优化：每次只依赖前两个状态，用两个变量滚动更新即可。

### 复杂度分析

- **时间复杂度：O(n)**
  - 只需一次遍历
- **空间复杂度：O(1)**
  - 只使用两个变量存储前两个状态
