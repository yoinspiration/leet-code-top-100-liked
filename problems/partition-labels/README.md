# Partition Labels

划分字母区间

## 题目要求

给你一个字符串 `s`，请你尽可能多地划分这个字符串，使同一字母**最多出现在一个分区中**。划分结果是连续的且覆盖整个字符串。

返回一个表示每个字符串片段长度的列表。

**示例：**

```
输入：s = "ababcbacadefegdehijhklij"
输出：[9,7,8]
解释：划分结果为 "ababcbaca"、"defegde"、"hijhklij"。
     每个字母最多出现在一个片段中。

输入：s = "eccbbbbdec"
输出：[10]
```

## 解法：贪心 [Greedy.cpp](Greedy.cpp)

### 算法思路

**第一步：** 记录每个字符在字符串中**最后出现的下标** `last[c]`。

**第二步：** 一次遍历，维护当前分区的起点 `start` 和右边界 `curEnd`：
- 对每个字符 `s[i]`，将 `curEnd` 延伸到 `max(curEnd, last[s[i]])`，确保该字符的所有出现都在同一分区内
- 当 `i == curEnd` 时，当前分区所有字符均已被包含，切割并记录长度

### 复杂度分析

- **时间复杂度：O(n)** — 两次线性遍历
- **空间复杂度：O(1)** — `last` 数组固定大小 26

### 解法评价

最优解法。与 Jump Game II 的思路高度相似：`curEnd` 随遍历不断向右延伸，到达边界时触发一次切割。
