# Largest Rectangle in Histogram

## 解法：单调栈 [Stack.cpp](Stack.cpp)

### 算法思路

用一个**单调递增栈**存储柱子下标（栈内高度从底到顶递增）。

当遍历到第 `i` 根柱子时，若它的高度小于栈顶柱子，说明栈顶柱子"向右再也无法延伸"，此时以栈顶柱子为高度计算最大矩形：

- **高度**：弹出的栈顶柱子高度 `h`
- **宽度**：`i - 新栈顶 - 1`（若栈空则宽度为 `i`，即左边无阻挡）

遍历结束时追加一根高度为 0 的哨兵柱，强制弹出栈内所有剩余柱子。

以 `[2,1,5,6,2,3]` 为例：

```
i=0 (h=2): st=[0]
i=1 (h=1): 1<2 → 弹0, h=2, w=1(栈空), area=2; st=[1]
i=2 (h=5): st=[1,2]
i=3 (h=6): st=[1,2,3]
i=4 (h=2): 2<6 → 弹3, h=6, w=4-2-1=1, area=6
           2<5 → 弹2, h=5, w=4-1-1=2, area=10
           st=[1,4]
i=5 (h=3): st=[1,4,5]
i=6 (哨兵0): 0<3 → 弹5, h=3, w=6-4-1=1, area=3
             0<2 → 弹4, h=2, w=6-1-1=4, area=8
             0<1 → 弹1, h=1, w=6(栈空),  area=6
结果: 10
```

### 复杂度分析

- **时间复杂度：O(n)**，每个下标最多入栈、出栈各一次
- **空间复杂度：O(n)**，栈最多存储 n 个下标
