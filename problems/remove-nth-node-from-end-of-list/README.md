# Remove Nth Node From End of List

删除链表的倒数第 N 个结点

## 题目要求

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例：**

```
输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

输入：head = [1], n = 1
输出：[]

输入：head = [1,2], n = 1
输出：[1]
```

## 解法：哑结点 + 快慢指针 [TwoPointers.cpp](TwoPointers.cpp)

### 算法思路

**核心思想：** 倒数第 `n` 个结点，与链表末尾的距离是固定的。让 `fast` 先走 `n + 1` 步（从哑结点算起），再让 `slow`、`fast` 同步前进，直到 `fast` 为空。此时 `slow` 正好停在「待删结点」的前驱，做一次 `slow->next = slow->next->next` 即可。

**为什么需要哑结点：** 若删除的是头结点（例如单结点链表删唯一结点），没有前驱；哑结点提供统一的前驱，避免对头结点单独分支。

**为什么 `fast` 先走 `n + 1` 步：** 从哑结点出发，间隔为 `n + 1` 时，`slow` 最终落在待删结点的前一个结点。

### 复杂度分析

- **时间复杂度：O(L)**，`L` 为链表长度，每个指针最多遍历一遍。
- **空间复杂度：O(1)**，只使用常数个指针（哑结点为栈上结点）。

### 解法评价

也可先遍历求长度再删，需两次扫描；快慢指针一次遍历，更简洁。本题不允许分配额外数组存结点地址时，双指针是常用写法。
