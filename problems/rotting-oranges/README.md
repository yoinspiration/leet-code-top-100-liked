# Rotting Oranges

腐烂的橘子

## 题目要求

给定一个 `m × n` 的网格，每格为：
- `0`：空格
- `1`：新鲜橘子
- `2`：腐烂橘子

每分钟，腐烂橘子会将四方向相邻的新鲜橘子也腐烂。返回直到所有橘子腐烂所需的最少分钟数；若不可能，返回 `-1`。

题目链接：[994. 腐烂的橘子](https://leetcode.cn/problems/rotting-oranges/)

## 解法：多源 BFS [BFS.cpp](BFS.cpp)

将所有初始腐烂橘子**同时**入队（多源），按轮次向外扩散——每轮处理队列当前所有结点，对应"同一分钟内同步腐烂"的语义。

**细节：**
- 预先统计新鲜橘子数 `fresh`，每腐烂一个 `-1`，最终判断是否为 0。
- 若初始 `fresh == 0`，直接返回 0。
- `minutes` 在每轮开始时 +1；最后一轮处理队列残余（无新腐烂产生），属于多余的一轮，故返回 `minutes - 1`。

- **时间复杂度：O(m×n)**，每个格子至多入队一次。
- **空间复杂度：O(m×n)**，队列最大容量。
