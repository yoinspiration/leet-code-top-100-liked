# Minimum Window Substring

最小覆盖子串

## 题目要求

给你一个字符串 `s` 和一个字符串 `t`。请你在 `s` 中找出包含 `t` 所有字符的最小子串。如果 `s` 中不存在这样的子串，返回空字符串 `""`。

**注意：**
- 子串必须包含 `t` 中所有字符，且次数要满足（多了可以，少了不行）
- 大小写敏感

**示例：**

```text
输入: s = "ADOBECODEBANC", t = "ABC"
输出: "BANC"

输入: s = "a", t = "a"
输出: "a"

输入: s = "a", t = "aa"
输出: ""
```

## 解法：滑动窗口 [SlidingWindow.cpp](SlidingWindow.cpp)

### 算法思路

使用「可变长度滑动窗口」：

1. 用 `need` 统计 `t` 中每个字符需要的数量
2. 用 `window` 统计当前窗口中每个字符的数量
3. 维护两个指针 `left` 和 `right` 表示当前窗口 `[left, right]`
4. 移动 `right` 扩大窗口，将字符加入 `window`
   - 如果某个字符在 `window` 中的数量刚好达到 `need` 的数量，则说明这个字符已经「满足要求」
5. 当所有字符都满足要求时（`formed == required`），开始移动 `left` 收缩窗口：
   - 每一步都尝试更新当前最小覆盖子串
   - 移除左边字符，如果导致某个字符不再满足 `need`，则窗口失效，退出收缩，继续扩展右边界

### 复杂度分析

- **时间复杂度：O(|s| + |t|)**
  - 初始化 `need`：O(|t|)
  - 滑动窗口遍历 `s`：每个字符最多被左右指针各经过一次
- **空间复杂度：O(C)**
  - `C` 为字符集大小，本题使用 ASCII 128，视为常数 O(1)

### 解法评价

- 典型的「最小子串覆盖」滑动窗口模板
- 与「无重复字符的最长子串」「找到字符串中所有字母异位词」等题目是同一类滑动窗口思路的变形
