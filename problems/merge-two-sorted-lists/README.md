# Merge Two Sorted Lists

合并两个有序链表

## 题目要求

将两个升序链表合并为一个新的升序链表并返回。新链表由拼接给定的两个链表的所有节点组成。

**示例：**

```
输入：list1 = [1,2,4], list2 = [1,3,4]
输出：[1,1,2,3,4,4]

输入：list1 = [], list2 = []
输出：[]

输入：list1 = [], list2 = [0]
输出：[0]
```

## 解法：哑结点 + 迭代 [Iterative.cpp](Iterative.cpp)

### 算法思路

**核心思想：** 使用哑结点（dummy）作为合并后链表的虚拟前驱，用指针 `tail` 始终指向当前结果链表的末尾；再用两个指针分别遍历 `list1`、`list2`，每次将较小（或相等时取左侧）的结点接到 `tail` 之后。

循环结束后，至少有一条链表已耗尽，将另一条剩余部分直接接到 `tail->next` 即可。

### 复杂度分析

- **时间复杂度：O(m + n)**  
  `m`、`n` 为两条链表长度，每个结点最多被访问一次。

- **空间复杂度：O(1)**  
  只使用常数个指针（哑结点为栈上结点，不额外分配链表结点）。

### 解法评价

迭代合并是标准写法，思路清晰且常数空间。也可用递归：比较头结点后将较小者 `next` 指向子问题的合并结果，但递归栈为 O(m + n)。
