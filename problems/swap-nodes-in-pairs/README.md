# Swap Nodes in Pairs

两两交换链表中的节点

## 题目要求

给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。  
你不能只是单纯地改变节点内部的值，而是需要实际进行节点交换。

**示例：**

```
输入：head = [1,2,3,4]
输出：[2,1,4,3]

输入：head = []
输出：[]

输入：head = [1]
输出：[1]
```

## 解法：哑结点 + 指针重连 [Iterative.cpp](Iterative.cpp)

### 算法思路

**核心思想：** 每次处理一对节点 `first`、`second`，把它们交换并接回原链表。

- 使用哑结点 `dummy`，统一处理头结点可能参与交换的情况。
- 指针 `prev` 指向当前待交换节点对之前的节点。
- 若 `prev->next` 和 `prev->next->next` 都存在，说明还有一对可交换：
  - `first = prev->next`
  - `second = first->next`
  - 重连顺序：
    - `first->next = second->next`
    - `second->next = first`
    - `prev->next = second`
- 交换后 `first` 成为这一段尾部，令 `prev = first` 进入下一轮。

### 复杂度分析

- **时间复杂度：O(n)**，每个节点最多访问常数次。
- **空间复杂度：O(1)**，只使用常数个指针变量。

### 解法评价

该写法满足题目“不改值，只换节点”的要求。递归也可实现，但会额外占用递归栈空间。
