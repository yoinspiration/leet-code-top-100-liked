# Binary Tree Maximum Path Sum

二叉树中的最大路径和

## 题目要求

给定一棵二叉树的根结点，返回其中**最大路径和**。

路径定义：从某一结点出发，沿父子结点连接，达到任意结点的序列。同一结点在路径中**最多出现一次**，路径不需要经过根结点。

题目链接：[124. 二叉树中的最大路径和](https://leetcode.cn/problems/binary-tree-maximum-path-sum/)

## 解法：后序 DFS [PostOrderDFS.cpp](PostOrderDFS.cpp)

定义辅助函数 `gain(node)`，返回以 `node` 为端点**向下单侧延伸**的最大路径和（即该结点能向父结点提供的最大贡献值）。

**核心思路：**
- 递归计算左、右子树的贡献值，负贡献直接视为 0（舍弃该侧）。
- 以当前结点为路径**最高点**时，路径和 = `left + node->val + right`，用于更新全局答案。
- 向父结点返回时只能选一侧：`node->val + max(left, right)`。

**为什么贡献值要取 max(0, ...)：** 若子树最大路径和为负，纳入只会使结果更小，不如从当前结点重新开始。

- **时间复杂度：O(n)**，每个结点访问一次。
- **空间复杂度：O(h)**，递归栈为树高；最坏 O(n)，平衡树 O(log n)。
