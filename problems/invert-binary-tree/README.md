# Invert Binary Tree

翻转二叉树

## 题目要求

翻转一棵二叉树，并返回其根节点。

## 解法一：递归 [Recursive.cpp](Recursive.cpp)

对每个非空结点：

1. 交换其 `left` 与 `right`；
2. 递归翻转左子树、右子树。

空结点直接返回。

- **时间复杂度：O(n)**，每个结点访问一次。
- **空间复杂度：O(h)**，递归栈为树高。

## 解法二：层序遍历（BFS）[BFS.cpp](BFS.cpp)

用队列从根开始逐层处理：每次弹出结点，交换其左右孩子，再将非空孩子入队。

- **时间复杂度：O(n)**
- **空间复杂度：O(w)**，队列中最多约一层宽度。

### 解法评价

递归最简短；BFS 不依赖递归栈，适合极深树时避免栈溢出（一般面试两种都掌握即可）。
