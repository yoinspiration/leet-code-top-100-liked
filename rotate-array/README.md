# Rotate Array

轮转数组

## 题目要求

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

**示例：**

```
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]

输入: nums = [-1,-100,3,99], k = 2
输出: [3,99,-1,-100]
解释:
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]
```

## 解法一：三次反转 [ThreeReversals.cpp](ThreeReversals.cpp)

### 算法思路

使用三次反转的方法，可以在 O(1) 额外空间内完成轮转：

1. **反转整个数组**
2. **反转前 k 个元素**
3. **反转后 n-k 个元素**

**原理：**
- 向右轮转 k 个位置，相当于将后 k 个元素移到前面
- 通过三次反转，可以巧妙地实现这个效果

**示例演示：**
```
nums = [1,2,3,4,5,6,7], k = 3

步骤1：反转整个数组
  [7,6,5,4,3,2,1]

步骤2：反转前 k=3 个元素
  [5,6,7,4,3,2,1]

步骤3：反转后 n-k=4 个元素
  [5,6,7,1,2,3,4] ✓
```

### 复杂度分析

- **时间复杂度：O(n)**
  - 三次反转，每次 O(n)
- **空间复杂度：O(1)**
  - 只使用了常数额外空间

### 解法评价

最优解法，时间复杂度 O(n)，空间复杂度 O(1)。思路巧妙，是解决数组轮转问题的经典方法。

## 解法二：使用额外数组 [ExtraArray.cpp](ExtraArray.cpp)

### 算法思路

1. 创建一个新数组存储结果
2. 对于原数组位置 `i`，新位置为 `(i + k) % n`
3. 将新数组复制回原数组

**关键公式：**
- 原数组位置 `i` 的元素，轮转后应该在位置 `(i + k) % n`

### 复杂度分析

- **时间复杂度：O(n)**
  - 遍历数组一次
- **空间复杂度：O(n)**
  - 需要额外的数组存储结果

### 解法评价

思路简单直观，但需要 O(n) 额外空间。适合理解问题，但在空间要求严格的情况下不如三次反转法。

### 示例演示

```
nums = [1,2,3,4,5,6,7], k = 3, n = 7

位置映射：
  原位置 i → 新位置 (i + k) % n
  0 → (0 + 3) % 7 = 3
  1 → (1 + 3) % 7 = 4
  2 → (2 + 3) % 7 = 5
  3 → (3 + 3) % 7 = 6
  4 → (4 + 3) % 7 = 0
  5 → (5 + 3) % 7 = 1
  6 → (6 + 3) % 7 = 2

结果：[5,6,7,1,2,3,4] ✓
```
