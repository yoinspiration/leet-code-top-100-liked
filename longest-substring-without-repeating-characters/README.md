# Longest Substring Without Repeating Characters

无重复字符的最长子串

## 题目要求

给定一个字符串 `s`，请你找出其中不含有重复字符的 **最长子串** 的长度。

**示例：**

```
输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

## 解法一：滑动窗口 [SlidingWindow.cpp](SlidingWindow.cpp)

### 算法思路

使用滑动窗口技巧，维护一个不包含重复字符的窗口：

1. 使用哈希表 `charIndex` 存储字符及其**最新出现的位置**
2. 使用两个指针 `left` 和 `right` 表示窗口的左右边界
3. 遍历字符串，`right` 指针向右移动：
   - 如果当前字符在窗口内已经存在（`charIndex[currentChar] >= left`），说明出现重复
     - 将 `left` 移动到重复字符的下一个位置：`left = charIndex[currentChar] + 1`
   - 更新字符的最新位置：`charIndex[currentChar] = right`
   - 更新最大长度：`maxLength = max(maxLength, right - left + 1)`

**关键点：**
- 使用哈希表记录字符的**最新位置**，而不是使用 set 记录字符是否出现
- 当遇到重复字符时，直接将左边界移动到重复字符的下一个位置，跳过中间的无效子串

### 复杂度分析

- **时间复杂度：O(n)**
  - 每个字符最多被访问两次（一次由 `right` 指针，一次由 `left` 指针）
- **空间复杂度：O(min(n, m))**
  - `m` 是字符集的大小（ASCII 最多 128，Unicode 最多 65536）
  - 哈希表最多存储 `m` 个字符

### 解法评价

最优解法，时间复杂度 O(n)，空间复杂度 O(min(n, m))。通过滑动窗口和哈希表的结合，高效地解决了问题。

## 解法二：暴力枚举 [BruteForce.cpp](BruteForce.cpp)

### 算法思路

双重循环枚举所有可能的子串，使用 set 记录每个子串中出现的字符，检查是否有重复。

### 复杂度分析

- **时间复杂度：O(n²)**
  - 外层循环：O(n)
  - 内层循环：O(n)
  - 每个子串的检查：O(n)
- **空间复杂度：O(min(n, m))**
  - set 最多存储 `m` 个字符（字符集大小）

### 解法评价

**注意：暴力枚举方法可能无法通过本题（会超时）。** 由于时间复杂度为 O(n²)，LeetCode 的测试用例中包含大规模数据，可能会导致超时。思路简单但效率低，仅作学习参考。实际提交时**必须使用滑动窗口优化版本**。
